### 全国测试

如果想看全国各地到搬瓦工的延迟分布，打开 `ping.chinaz.com`，输入上面的 IP，就能看到全国各省的 ping 情况。北方电信跑出来跟南方移动可能差 20–30ms，很正常。

ping 默认只测一次延迟，测丢包的话 Windows 上可以用 `ping -n 50 89.208.246.192`，发 50 个包看有没有丢。好的 CN2 GIA 线路丢包应该在 1% 以下，超过 5% 就说明线路状态不太好。

## 晚高峰表现：最重要的指标没人告诉你

延迟这件事有个坑：白天测出来 140ms，不代表晚上 8–11 点也是这个数。

普通国际线路在晚高峰经常会涨到 300ms 以上，甚至出现大范围丢包。原因是出国带宽不够，高峰期拥堵。

CN2 GIA 做的事情本质上是：给你的流量打上"高优先级"标签，就算带宽拥堵也不会被随意丢掉。用 DC9 实测下来，晚上 9 点的延迟跟下午 3 点几乎没差，稳定性这点是最值钱的。
